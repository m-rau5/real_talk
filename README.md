# Real Talk

A Full-Stack Social Media app with posting and friend functionality made in JavaScript using the MERN stack.
## Tech Stack - MERN

**Client:** React JS

**Server:** Node JS, Express JS

**Database:** Mongo DB
  
## To Run Locally
  
  Clone the project

```bash
  git clone https://github.com/m-rau5/real_talk.git
```

Go to the project directory

```bash
  cd ./real_talk
```

Install required dependencies

```bash
  npm install
```

```bash
  cd ./client
  npm install
```

Start the server

```bash
  cd ./server
  npm run start
```
Start the client

```bash
  //open a new terminal
  cd ./client
  npm start
```
